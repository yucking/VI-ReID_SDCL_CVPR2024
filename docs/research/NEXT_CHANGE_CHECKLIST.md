# Next Change Checklist

Before the next code change, answer:

1. What is the log evidence for the current bottleneck?
2. Is this a Stage-1 or Stage-2 problem?
3. Why can existing SDCL/CNL/CRA modules not solve it?
4. What is the smallest implementation of the new mechanism?
5. Which intermediate metric should improve first?
6. Which result would falsify the mechanism?
7. Is there only one primary variable?
8. Has the baseline been repeated?
9. Has independent implementation review completed?
10. Is the evidence strong enough to justify full training?

If any answer is `UNKNOWN`, do not start a full experiment.
